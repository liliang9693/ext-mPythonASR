# 讯飞语音账号获取流程
- 首先到讯飞开放平台注册并登录： https://www.xfyun.cn/
- 进入**控制台**，**创建应用**，点开创建的应用就可以得到**服务接口认证信息**也就是**APPID、APISecret、APIKey**，填入对应积木中即可。
